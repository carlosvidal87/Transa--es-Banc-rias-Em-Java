<div></div>
===== Menu Principal =====<br>
1. Transferência entre contas<br>
2. Depósito numa conta<br>
3. Retirada de uma conta<br>
4. Mostrar log de transações<br>
5. Criar Cliente e Conta<br>
6. Mostrar Clientes<br>
7. Sair
